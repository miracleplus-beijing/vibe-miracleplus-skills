# JSON-Canvas

**类型**: 评测Skill

**评分**: 5/5

**领域**: 产品开发

**描述**: Create and edit JSON Canvas files (.canvas) with nodes, edges, groups, and connections

## 主要功能
- 创建和编辑JSON Canvas文件(.canvas)
- 支持4种节点类型: text, file, link, group
- 创建节点之间的连接(edges)
- 支持颜色、标签、背景样式等配置
- 用于Obsidian和其他应用的无限画布

## 节点类型
- Text: Markdown内容
- File: 文件/附件引用
- Link: 外部URL
- Group: 视觉容器

## 支持的功能
- 节点分组和组织
- 边的连接和标签
- 颜色预设(1-6)和十六进制颜色
- 背景样式: cover, ratio, repeat

## 文件格式
JSON格式，包含nodes和edges数组
