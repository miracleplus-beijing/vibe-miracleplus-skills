# Literature-Review

**类型**: 评测Skill

**评分**: 5/5

**领域**: 市场调查

**描述**: Conduct comprehensive, systematic literature reviews using multiple academic databases

## 主要功能
- 系统性文献综述
- 多数据库搜索(PubMed, arXiv, bioRxiv, Semantic Scholar等)
- 文献筛选和质量评估
- 数据提取和综合
- 引文验证
- 生成markdown和PDF格式的专业文档

## 工作流程
1. Phase 1: 规划和范围界定
2. Phase 2: 系统文献搜索
3. Phase 3: 筛选和选择
4. Phase 4: 数据提取和质量评估
5. Phase 5: 综合和分析
6. Phase 6: 引文验证
7. Phase 7: 文档生成

## 支持的数据库
- PubMed/PMC (via gget)
- bioRxiv/medRxiv (via gget)
- arXiv
- Semantic Scholar
- 专业数据库(ChEMBL, UniProt, KEGG等)

## 依赖要求
- Python requests
- pandoc
- xelatex

## 引文格式
APA, Nature, Vancouver, Chicago, IEEE
