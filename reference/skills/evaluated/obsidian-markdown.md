# Obsidian-Markdown

**类型**: 评测Skill

**评分**: 4/5

**领域**: 产品开发

**描述**: Create and edit Obsidian Flavored Markdown with wikilinks, embeds, callouts, properties, and other Obsidian-specific syntax

## 主要功能
- 创建和编辑Obsidian风格的Markdown
- 支持wikilinks内部链接
- 嵌入文件、图片、音频、PDF
- Callouts(提示框)
- 前置属性(frontmatter)
- 标签系统
- Mermaid图表
- LaTeX数学公式
- 脚注和注释

## 支持的Markdown特性
- 基本格式(粗体、斜体、删除线、高亮)
- 列表(有序、无序、任务列表)
- 表格
- 代码块
- 引用
- 水平线

## Obsidian特定功能
- Wikilinks: [[Note Name]], [[Note#Heading]]
- 嵌入: ![[file]], ![[image.png|300]]
- Callouts: > [!note], > [!warning]等
- 属性: YAML frontmatter
- 标签: #tag, #nested/tag
